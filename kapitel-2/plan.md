# Modulplan: bilder och layout-introduktion

Kapitlet fördjupar arbetet med bilder och introducerar hur man positionerar element med marginaler. Eleverna bygger en reklamsida med bakgrundsbilder och textplacering.

## Mål

- Använda `<figure>` och `<figcaption>` för bilder med bildtext.
- Positionera element med `margin-left` och `margin-top`.
- Använda `text-transform` för versaler.
- Styra bakgrundsbilders position med `background-position`.
- Styla länkar utan understrykning (`text-decoration: none`).

## Förkunskaper

- Kapitel 1: HTML-grundstruktur, `<h1>`, `<p>`, `<a>`, `<img>`, grundläggande CSS (färg, typsnitt, bakgrund).

## Nya begrepp

### HTML
`<figure>`, `<figcaption>`

### CSS
`margin-left`, `margin-top`, `text-transform: uppercase`, `background-position`, `text-decoration: none`, `padding` (på inline-element)

## Pedagogiska grundregler

- Visuellt engagerande tema (Fiji-reklam med tropiska bilder).
- Fokus på att flytta och placera element — en brygga till mer avancerade layouter i kapitel 3.
- Repetition av kapitel 1-koncept (bakgrundsbilder, färger, typsnitt).

## Lektioner

### Lektion 1: reklamsida med bilder och marginaler (mappen: fiji-reklam)
- **Innehåll:** `margin-left`, `margin-top`, `text-transform: uppercase`, `background-position`, stilade länkar med `padding` och `text-decoration: none`.
- **Mål:** kunna placera text och element med marginaler, styla länkar som knappar.
- **Repetition:** `<h1>`, `<h2>`, `<p>`, `<a>`, `background-image`, `color`, `font-family` från kapitel 1.

### Lektion 2: porträtt med figurer (mappen: portratt)
- **Innehåll:** `<figure>`, `<figcaption>`, bildlayout.
- **Mål:** använda `<figure>` för semantiskt korrekt bildhantering.
- **Repetition:** `<img>`, CSS-styling från kapitel 1.
- **Status:** påbörjad men ej slutförd.

## Vanliga misstag

- Förväxlar `margin` (yttre avstånd) och `padding` (inre avstånd).
- Felaktiga sökvägar till bilder i `bilder/`-mappen.
- Glömmer att `margin-top` på ett första element kan kollapsa med `body`.
- Skriver `text-transform: Uppercase` istället för `text-transform: uppercase`.
