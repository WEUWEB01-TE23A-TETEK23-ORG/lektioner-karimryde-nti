# Dagbok - Projekt: Min sida om något jag gillar

I den här lektionen introducerade vi det första stora projektet där vi knyter ihop grunderna i HTML och CSS. Vi går från små kodexempel till att bygga en sammanhängande undersida.

### Nya begrepp och verktyg
- **Scaffold (Mall):** En färdig grundstruktur som man utgår ifrån för att snabbare komma igång.
- **HTML-tabeller:** Ett sätt att organisera data i rader (`<tr>`), rubrikceller (`<th>`) och dataceller (`<td>`).
- **Google Fonts:** Hur vi hämtar snygga typsnitt utifrån för att ge sidan karaktär.

### Kodexempel: Tabellstruktur
```html
<table>
    <tr>
        <th>Egenskap</th>
        <th>Information</th>
    </tr>
    <tr>
        <td>Kategori</td>
        <td>Mina favoriter</td>
    </tr>
</table>
```
